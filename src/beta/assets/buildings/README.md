# Sprites des bâtiments — Carte Patrimoine (beta)

Dépose ici tes images générées (Midjourney / DALL-E / Ideogram).

## Règles de nommage (IMPORTANT)

Le fichier **doit** porter exactement le nom du `catalogId`, en `.png` ou `.webp`.
Le chargement est automatique : dès qu'un fichier au bon nom est présent,
il remplace l'icône dans la carte. Aucun code à toucher.

| Fichier à déposer        | Investissement              |
|--------------------------|-----------------------------|
| `livret.png`             | Livret A                    |
| `assurance_vie.png`      | Assurance Vie — Fonds Euros |
| `obligations_etat.png`   | Obligations d'État          |
| `bourse_etf.png`         | Bourse — ETF World          |
| `or_metaux.png`          | Or & Métaux précieux        |
| `crowdfunding_immo.png`  | Crowdfunding Immobilier     |
| `scpi.png`               | Parts de SCPI               |
| `produit_structure.png`  | Produit Structuré           |
| `crypto.png`             | Crypto-actifs (BTC / ETH)   |
| `business.png`           | Business Automatisé         |
| `parking.png`            | Parking / Box               |
| `lmnp.png`               | Location Meublée (LMNP)     |
| `immo_classique.png`     | Locatif Classique           |
| `club_deal_immo.png`     | Club Deal Immobilier        |

## Format recommandé

- **Taille** : 256×256 px (carré)
- **Fond** : transparent (PNG/WebP)
- **Cadrage** : le bâtiment centré, vue isométrique 45°
- Pas besoin d'ombre — le code ajoute le cadre lumineux automatiquement

Les fichiers manquants gardent l'icône Lucide actuelle en repli.
